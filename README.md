# 📈 Stock Price Prediction Using Machine Learning

## 📌 Overview

This project focuses on predicting short-term stock closing prices using Machine Learning models in Python. Historical stock market data was collected using the yfinance library, and regression models were trained to forecast the next day’s closing price.

---

## 🎯 Objective

* Collect historical stock market data
* Perform feature engineering on stock data
* Predict next-day closing prices
* Compare machine learning model performance
* Visualize actual vs predicted stock prices

---

## 🛠️ Tools Used

* Python
* pandas
* numpy
* matplotlib
* seaborn
* scikit-learn
* yfinance

---

## 📊 Dataset

Historical stock market data was fetched using Yahoo Finance API through the yfinance library.

### Features Used:

* Open Price
* High Price
* Low Price
* Close Price
* Volume

### Stocks Analyzed:

* Apple (AAPL)
* Tesla (TSLA)
* Microsoft (MSFT)
* Google (GOOGL)
* Amazon (AMZN)

---

## 🤖 Machine Learning Models

The following regression models were implemented:

* Linear Regression
* Random Forest Regressor

---

## 🔍 Analysis Performed

* Historical stock data collection
* Data preprocessing and cleaning
* Feature engineering
* Train-test split for prediction
* Model training and evaluation
* Actual vs predicted price visualization
* Performance comparison between models

---

## 📈 Evaluation Metrics

The models were evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

---

## 🧠 Key Insights

* Historical stock data was successfully collected using yfinance
* Features like Open, High, Low, Close, and Volume improved prediction
* Random Forest performed better than Linear Regression
* Predicted prices followed actual market trends closely
* Visualization helped compare prediction accuracy effectively

---

## ✅ Conclusion

This project demonstrated how machine learning can be applied to short-term stock price prediction using historical financial data. Among the implemented models, Random Forest provided better prediction performance and handled stock market fluctuations more effectively.
